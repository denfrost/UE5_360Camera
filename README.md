# UE5_360Camera
This is a Stereo 360 Camera on UE5.
<br />
It can generate:
Mono/Stereo equirectangular image in:
- Final Color
- Base Color
- Roughness
- Scene Depth
- World Normal
- Metallic
- Ambient Occlusion

## Examples
The data was generated in a self-built island scene.
